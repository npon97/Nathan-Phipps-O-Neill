Binaural Sensory Augmentation

This project was a sole project for the requirements of my masters degree in electronic and computer engineering. The fundamental goal / principal of the project was to build a device that could be used as a navigational aid by using non-linguistic spatial auditory cues.
The pictures below show a few key components of the prototype:

  1. The main compute module (Raspberry Pi in a green case)
  2. The Lithium Ion battery (black block)
  3. The GPS/GNSS/GLONASS module used for location retrieval (inside the red case)
  4. The Magnetometer used for direction retrieval (inside the red case too)
  5. The headphones used to output spatial audio to the user
The compute module would retrieve input from the GPS module and direction information from the magnetometer module and use it to render spatial audio using the OpenAL library. This library is usually used for games/virtual environments, but in this case, it's used in real life. Then the spatial audio is output to the headphones.
